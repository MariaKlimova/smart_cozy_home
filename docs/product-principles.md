# Продуктовые принципы

## Метафора

- **Home Assistant** — «электрощиток дома»: стабильность, automations, entities.
- **Это приложение** — «ощущение дома»: спокойствие, понятность, забота.

## UX

1. **Один главный экран** — состояние дома (Home State). Остальное — табы без глубокой вложенности.
2. **Максимум 1 уровень глубины** на основных экранах.
3. **3–5 ритуалов** на главном пути; расширенная настройка скрыта.
4. **Никаких entities** в пользовательском интерфейсе.
5. **Никакого YAML** и графа automations в UI по умолчанию.

## Copy voice

- Спокойный, тёплый, без восклицаний.
- Формулировки про жизнь: «Спокойный вечер», «Ты дома», не «light.living_room turned on».
- Мягкие предложения вместо алертов: «Похоже, в спальне темно — включить свет?»

## Визуал

- Calm, warm, minimal.
- Крупные зоны нажатия (минимум 48dp).
- Редкие анимации: fade, без bounce.
- Тёплая палитра (см. `apps/mobile/src/theme/tokens.ts`).

## Действия

- Крупные, редкие, уверенные.
- Ритуалы вместо automations в языке продукта.
